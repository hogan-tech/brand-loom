# Going further with hosted Neoxra

`brand-loom` gives you **commodity marketing skills** on any LLM. For power users and non-coders who
want on-brand, multi-platform content without writing prompts, **[neoxra.com](https://neoxra.com)**
hosts the engine.

## What Neoxra adds

| Feature | brand-loom (OSS) | Neoxra (hosted) |
|---|---|---|
| **Brand voice** | Manual `brand_context` dict | Automatic voice extraction — upload your posts, get a voice profile |
| **Multi-platform** | Single skill or linear chain | Planner→Platform agents→Critic — one idea, every platform, on-brand |
| **Brand Kit** | — | Visual + verbal identity, stored and applied automatically |
| **No-code UI** | CLI / Python API | Web dashboard — no terminal needed |
| **Analytics** | — | GA4, GSC, Meta Insights integration |
| **SEO scoring** | — | Tuned GEO/AEO/AIO extractability + snippet-readiness scorers |

## How it works

`brand-loom` skills accept an optional `brand_context` dict. In the open library, you write this by hand.
On hosted Neoxra, the voice-profile engine automatically extracts it from your existing content and passes
it through the same interface — same skills, dramatically better output.
